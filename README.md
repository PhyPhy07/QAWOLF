# QAWOLF

QA Wolf take-home assignment (QA Engineer). The runnable project and full write-up live in **`[qa_wolf_take_home](./qa_wolf_take_home/)`**.

## Quick start

```bash
cd qa_wolf_take_home
npm install
node index.js
```

- **Headless (no browser window):** `npm run present` or `HEADLESS=1 node index.js`
- **Browsers:** `npm install` runs `playwright install chromium` via `postinstall`. If you used `--ignore-scripts`, run `npm run install:browsers` once.
